## Contexto

Este proyecto corresponde a un examen de **Programación Web**, cuyo objetivo es evaluar las habilidades del estudiante en el desarrollo de un **frontend funcional** capaz de interactuar con una **API REST**.

Se busca simular un entorno real de desarrollo web, donde el frontend consume datos desde un servidor y los presenta de manera dinámica al usuario.

---

## Objetivo

Desarrollar una aplicación web básica que:

- Consuma datos desde una API
- Procese la información obtenida
- Muestre los datos dinámicamente en pantalla

---

## Competencias Evaluadas

- Conexión a una API REST usando `fetch()`
- Manejo de datos en formato JSON
- Manipulación del DOM con JavaScript
- Creación de contenido dinámico en HTML
- Aplicación de estilos con CSS
- Organización y buenas prácticas del código

---

## Tecnologías Utilizadas

- **HTML** → Estructura de la página  
- **CSS** → Diseño y estilos  
- **JavaScript** → Lógica e interacción  
- **API REST** → Fuente de datos  

---

## Descripción del Proyecto

La aplicación web desarrollada debe:

1. Realizar una petición a una API externa  
2. Obtener los datos utilizando `fetch()`  
3. Procesar la información recibida  
4. Generar contenido dinámico en el HTML  
5. Mostrar los datos de forma clara y organizada  
6. Aplicar estilos para mejorar la presentación  

---

## Criterios de Evaluación

- Correcto consumo de la API  
- Funcionamiento del código JavaScript  
- Estructura adecuada del HTML  
- Diseño visual con CSS  
- Claridad en la presentación de datos  
- Buenas prácticas de programación
